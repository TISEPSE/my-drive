# My Drive Client

This is the frontend component of the My Drive application, built with React and Vite.

## Features

- **Dashboard**: Overview of connected accounts and recent activity.
- **File Explorer**: Browse files with support for different view modes.
- **Modals**: Handle file sharing, moving items, and detailed views.
- **Authentication**: Integrated login and account selection.
- **Toasts**: Real-time feedback for file uploads and actions.

## Tech Stack

- **React**: UI library.
- **Vite**: Build tool and dev server.
- **Context API**: For state management (Auth, Theme, Toast, Uploads).
- **Vitest**: Testing framework.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Run tests:
   ```bash
   npm run test
   ```
